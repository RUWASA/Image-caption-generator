# Image Caption Generator

## Overview
An AI-powered application that generates descriptive captions for images using the **BLIP (CNN + LSTM)** model. This tool combines deep learning with NLP for automatic captioning.

## Technologies Used
Python, Streamlit, Hugging Face Transformers, PyTorch, CNN, LSTM

## Features
- Generates accurate captions for uploaded images.
- Preprocessing and caption enhancement using Hugging Face Transformers.
- User-friendly interactive interface via **Streamlit**.

Usage

Upload an image and get a generated caption instantly.

Preview multiple suggested captions for enhanced usability.
